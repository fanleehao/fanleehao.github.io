Hexo 搭建过程及采坑点

安装教程

  • git
  • nodejs

使用

  • hexo init [dirname]
  • cd [dirname]
  • npm install
  • hexo new
  • hexo g
  • hexo s
  • npm install hexo-deployer-git –save
  • hexo d

新建博客

  • hexo new “blog name”
  • 在本地./source/_posts文件夹下出现blogname.md文件，编辑文本和主要内容
  • hexo clean
  • hexo g
  • hexo d
